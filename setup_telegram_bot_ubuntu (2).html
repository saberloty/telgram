📌 نصب ربات تلگرام روی Ubuntu 22.04

1. به‌روزرسانی سیستم:

sudo apt update && sudo apt upgrade -y

2. نصب Python 3 و pip:

sudo apt install python3 python3-pip -y

3. ساخت پوشه برای ربات:

mkdir ~/mybeautybot
cd ~/mybeautybot

4. ساخت فایل ربات:

nano bot.py

کد ربات را داخل این فایل جای‌گذاری کنید.

5. نصب کتابخانه‌های موردنیاز:

pip3 install python-telegram-bot==13.15
pip3 install pytz certifi tornado cachetools APScheduler

6. اجرای ربات:

python3 bot.py

7. اجرای دائم با screen (اختیاری):

sudo apt install screen -y
screen -S beautybot
python3 bot.py

برای خروج از سشن بدون توقف ربات:

Ctrl + A سپس D

برای بازگشت به سشن:

screen -r beautybot
نکته مهم:
در فایل bot.py این مقادیر را طبق اطلاعات خود وارد کنید: 
TOKEN = "توکن ربات"
ADMIN_ID = 123456789
CHANNEL_ID = -100xxxxxxxxxx

با آرزوی موفقیت 🚀
